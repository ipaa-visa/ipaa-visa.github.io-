# [ipaa-visa.github.io](https://github.com/ipaa-visa/ipaa-visa.github.io)

IPAA（外国人入管手続研究会）は、東海地域において国際業務を専門とする行政書士有志により構成された任意団体です。

# Features
 
当団体では、日頃より外国人関連の国内外の法令はもとより、各種制度や実務運用について継続的に調査・研究を行い、正確かつ最新の情報の共有・提供に努めております。
